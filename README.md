# Bob Run

Bob Run is a small but **fun** Unity game.

## Repository contents

- Only some project files are included in this repository because some Unity files are too large for GitHub. 
- The full Unity project and a ready-to-play build are available under the Releases section. 
- The installation script (`BobRun.iss`) is included in this repository. 

## Download and installation

1. Go to the **Releases** section of this repository. 
2. Download the `BobRun Setup.exe`

If you notice any bug, please contact lazyloopstudio@gmail.com

Enjoy playing Bob Run!
